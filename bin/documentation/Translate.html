LaTEX->XML

Translate button starts translating document.

After pushing this button, program will start checking the document for structure(right references, figure structure etc).

If the structure of the document is not right it will finish working out the document and show error;

There are two types of errors:

      1. Figure declaration missing

                       

This error will be showed if the document will not have right figure declaration structure.

Right structure:

\begin{figure}

...content of the figure....

\end{figure}


       Error structure:

Any other structure that declares figures without using LaTEX tags (\begin{figure}...\end{figure})


Example:

       Error structure:


{\bf Figure 1 - Name of figure \\

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. \\


Right structure:

\begin{figure}

{\bf Name of figure }

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content. Figure content.

\end{figure}

2.  Supporting data declaration missing


Error structure:

Any other structure that declares figures without using LaTEX tags (\begin{suppfigure}...\end{suppfigure})


Example:

       Error structure:


\paragraph{Figure S1}


.....Content....


Right structure:

\begin{suppfigure}

{\bf Figure S1 }
