LaTEX->XML

'Choose folder' button opens explorer in which you should choose folder with files that you want to translate.

The folder should contain two files (document itself)LaTEX and (frontmatter)xml, if it contains any bibliography file it will display that.

If you choose the folder, that doesn't contain all necessary files, it will show you the warning message "Chosen folder doesn't contains all necessary files".

After choosing folder with files it will show file names

If folder contains a bibliography file program will display it's name.

After loading all files, program displays LaTEX document in the document window.

In help console the program writes chosen directory path and file names.
